CLI

Install the Sensei CLI

One command gets you the CLI and everything you need to initialize repos, run the server, and set up MCP.

🍺 Homebrew macOS / Linux
brew install mizukisu/tap/sensei

Updates automatically with brew upgrade

📦 npm / bun All platforms
bun install -g sensei
# or
npm install -g sensei

Quick start

sensei init          # initialize your repo
sensei serve         # start local server + graph indexer
sensei setup --mcp   # register MCP server for Claude Code
What sensei does

The observability layer for AI-assisted development

Like Code Climate for code quality — sensei gives you the metrics to understand and improve how your team uses AI.

🎯

First-Try-Right Scoring

Track how often AI completes tasks on the first attempt. Per-repo, per-agent, per-team. Trending over time.

💰

Cost Intelligence

Token-level cost per task, session, and repo. See exactly where money is spent and what drives cache hit rates.

📦

Context Packing

Smart token-budgeted context using BM25 + semantic search. Right code, right size, every request.

🔄

Session Continuity

Automatic crash recovery and progress snapshots. Never lose context mid-task.

📚

Doc Drift Detection

Traceability matrix linking code to docs. Flags stale documentation automatically when code changes.

🧩

Plugin Ecosystem

Skills, commands, and hooks that inject guardrails into every Claude Code session. Zero-friction.
